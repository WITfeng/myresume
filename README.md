# resume

> A Vue.js project

## Build Setup

``` bash
#clone the repository
git clone https://github.com/WITfeng/resume.git
cd resume

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
